以下は ダミー縦長答案画像 を使った実際の GUI キャプチャです。

1. 問題一覧画面（採点モード選択プルダウン付き）

問題一覧画面に紫色の帯で「採点モード」プルダウンを追加しました。
「1枚ずつ」または「一覧（グリッド）」を事前に選択してから採点を開始できます。

2. 採点画面（〇/×ボタン + フィルタ + 未採点カウント）

右パネルに以下を追加:
- 〇 正解ボタン（満点付与）/ × 不正解ボタン（0点付与）
- キーボード: m=〇正解 b=×不正解
- 「未採点のみ表示」チェックボックス
- 未採点カウント表示

3. 〇正解ボタン押下後（背景フィードバック: 薄青）

〇ボタン（m キー）で満点を付与すると、キャンバス背景が薄青(#E3F2FD)に
一瞬フラッシュし、400ms 後に白に戻ります。直感的に「正解」と分かります。

4. ×不正解ボタン押下後（背景フィードバック: 薄赤）

×ボタン（b キー）で0点を付与すると、キャンバス背景が薄赤(#FFEBEE)に
フラッシュします。

5. 中間点入力（背景フィードバック: 薄橙）

数値キー等で中間点を入力すると、薄橙(#FFF3E0)にフラッシュします。

6. 「未採点のみ表示」フィルタ ON

チェックを入れると、既に採点済みの生徒をスキップして
未採点の答案のみを ←→ で移動できます。

7. 全員採点済み状態

全員の採点が完了すると、未採点カウントが 0 になります。

8. 一覧（グリッド）モード

モードバーで「一覧」に切り替えると、全生徒をサムネイルで一覧表示。
得点ボタンを選択してからサムネイルをクリックで採点できます。
