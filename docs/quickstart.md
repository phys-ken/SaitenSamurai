# クイックスタート

最初の採点を **5 分** で体験するためのガイドです。

---

## 事前準備

採点を始める前に、次の素材を用意してください。

| 必要なもの | 説明 |
|---|---|
| **スキャン画像** | 答案用紙をスキャンした JPEG / PNG / PDF |
| **Mark2 座標ファイル**（マーク採点時） | Mark2 テンプレートサイトからダウンロードした座標 Excel ファイル |

!!! tip "まずはサンプルで試す"
    リポジトリの `sample_basefile/` フォルダに、すぐ試せるサンプルファイルが入っています。

!!! info "解答用紙の作り方"
    テスト用紙の自作やカスタマイズ方法は **[解答用紙の準備](preparation.md)** をご覧ください。

---

## Step 1: モードを選択して起動

`SaitenSamurai.exe` をダブルクリックすると、モード選択ダイアログが表示されます。

試験の内容に合わせてモードを選んでください。

![モード選択ダイアログ](images/01_startup_mode_dialog.png){ .screenshot-small }
<span class="caption">起動時のモード選択画面</span>

| モード | こんなときに |
|---|---|
| **マーク採点** | マークシートのみの試験 |
| **記述式採点** | 記述式のみの試験 |
| **マーク＋記述** | マーク式と記述式が混在する試験 |

---

## Step 2: ファイルを読み込む

モードを選択すると、メイン画面が表示されます。

=== "マーク採点"

    1. **座標ファイル**（Mark2 形式の Excel）を選択
    2. **スキャン画像フォルダ**を指定
    3. **OMR 認識を実行** — 正答データ（`answer_key.xlsx`）が自動生成されます

    ![マーク採点メイン画面](images/02a_main_mark_only.png){ .screenshot }
    <span class="caption">マーク採点モードのメイン画面</span>

=== "記述式採点"

    1. **スキャン画像フォルダ**を指定
    2. 採点領域をマウスで設定

    ![記述式メイン画面](images/02c_main_descriptive_only.png){ .screenshot }
    <span class="caption">記述式採点モードのメイン画面</span>

=== "マーク＋記述"

    1. **座標ファイル** → **スキャン画像フォルダ**の順に指定
    2. **OMR 認識を実行**
    3. 記述式の領域を追加設定

    ![混合モードメイン画面](images/02b_main_mark_descriptive.png){ .screenshot }
    <span class="caption">マーク＋記述モードのメイン画面</span>

---

## Step 3: 採点して結果を出力

採点が完了したら、結果を出力します。

出力先のフォルダには以下が自動生成されます。

```
_saiten_grading_results/
├── 00_Processing/      ← 中間処理ファイル
├── 01_Results/         ← OMR データ・正答ファイル
├── 02_Graded_Detail/   ← 採点済み答案画像（○×マーク付き）
└── 03_Final_Report/    ← 成績サマリー・統計レポート・CTT分析
```

描画設定をカスタマイズして、○×マークや点数の表示位置を調整することもできます。

![描画設定画面](images/03_rendering_settings.png){ .screenshot }
<span class="caption">描画設定のカスタマイズ</span>
